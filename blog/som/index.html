2025-05-29

La asignatura de Sistemas Operativos Monopuesto tiene como objetivo que el alumnado aprenda a instalar, configurar, administrar y mantener sistemas operativos en ordenadores individuales (monopuesto), tanto de tipo Windows como Linux.

Se enseña desde cero cómo preparar un equipo para su uso, incluyendo instalación de software, configuración del sistema, gestión de usuarios, seguridad básica y más.
